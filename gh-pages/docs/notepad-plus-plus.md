# Notepad++ Notes

## Style Configurator Settings (custom color settings)

- The Main Styles file ([stylers.xml](https://gist.github.com/Ross-Thanscheidt/b8fd9622a6a7626ccaa7bfef83d2631f#file-stylers-xml)) goes to `$env:AppData\Notepad++\`
- The [DarkModeDefault.xml](https://gist.github.com/Ross-Thanscheidt/b8fd9622a6a7626ccaa7bfef83d2631f#file-darkmodedefault-xml) theme file goes to `$env:AppData\Notepad++\themes\`
